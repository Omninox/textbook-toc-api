Instructions:

  • Include section numbers in the SECOND COLUMN
  • The FIRST COLUMN should have a chapter number (numerical- 1, 2, 3, 4, 5, 6). To indicate that it is a chapter title and not a section title, label the second column 0
  • If a textbook has units or parts, use a non-integer identifier in the FIRST COLUMN (like I, Part II, Unit C, Unit 3
  • The THIRD COLUMN should have the title of the chapter, section, part, or unit. Remember, if it's a unit or part, use a non-integer value in the first column. If it's a chapter, use an integer value. If it's a section, include the chapter number in the first column and the section identifier (1.1, 3.4, etc.) in the second column. Titles always go in the third column.
